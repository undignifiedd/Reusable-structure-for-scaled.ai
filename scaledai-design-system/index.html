House rules

Three things not to lose in a rewrite

Global tracking. The * { letter-spacing: 0.01em } rule is easy to drop and its absence is immediately visible. Keep it in the base layer.
Embed mode. The chat widget renders inside a host page and must not paint its own background — html.widget-embed-mode forces transparent. Carry it into any new embeddable surface.
No dark mode yet. Every token is light-only — there is no .dark block. If a property ships a dark theme, it's doing it outside this system. Decide before extending.
/* consume tokens — never hardcode hex in components */
.button {
  background: hsl(var(--primary));
  color:      hsl(var(--primary-foreground));
  border-radius: var(--radius-md);      /* 16px */
  box-shadow: var(--shadow-sm);
}
.button:focus-visible {
  box-shadow: 0 0 0 3px hsl(var(--ring) / .4);
}
